# MATHS BEYOND LIMITS BALKANS (MBLB)
- Flutter app for scheduling events, scanning QR into PostgreSQL database, and logging features
- Coded fully in Dart, C and PostgreSQL
- Dynamically linked

# HOW TO RUN?
```
flutter doctor
flutter pub clean
flutter pub get add
flutter run
```
